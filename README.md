# Cat Energy

**Cat Energy** — адаптивный лендинг бренда функционального питания для котов.  
Проект выполнен по макету из [Figma](https://www.figma.com/design/gqM9B9ZUTgcmVXsQ2W7V8C/%D0%9A%D1%8D%D1%82%D1%8D%D0%BD%D0%B5%D1%80%D0%B4%D0%B6%D0%B8---landing-Copy-?node-id=2403-436&t=MrcbOXLIoeKLMGxM-0) и демонстрирует современные подходы к вёрстке, адаптивности и интеграции внешних API без использования сторонних библиотек и фреймворков.

---

## Описание проекта

Лендинг состоит из нескольких ключевых секций:

- **Главный экран (Hero)** — заголовок, подзаголовок и CTA-кнопка.
- **Программы питания (Slim / Pro)** — карточки с описанием и ссылками.
- **Живой пример** — интерактивный слайдер “до / после”, реализованный на чистом JavaScript.
- **Карта** — встроенная Яндекс.Карта через официальное API.

Цель проекта — продемонстрировать:

- адаптивную и кросс-браузерную вёрстку;
- семантический HTML и структуру по методологии **БЭМ**;
- реализацию интерактивных элементов без сторонних библиотек;
- интеграцию **Яндекс.Карт API**.

---

## Основные особенности

- **Адаптивная вёрстка** для мобильных, планшетов и десктопов
- **Слайдер “Живой пример”** реализован на чистом JavaScript
- **Интеграция Яндекс.Карт API**
- **Семантический HTML + БЭМ**
- **SCSS (SASS)** для организации и переиспользования стилей
- **Кросс-браузерность:** Chrome, Firefox, Safari, Edge

---

## Технологии

- HTML5
- SCSS (SASS)
- JavaScript (без библиотек)
- Flexbox / Grid
- Методология БЭМ
- Git / GitHub Pages
- Yandex.Maps API
